# Instructor Dashboard Page

## Overview
Dashboard for instructors to manage their courses and students.

## Functionality

- View all published courses
- View student list and progress
- View submissions and grades
- Course analytics and statistics
- Create new courses
- Course settings and editing

## Dashboard Sections

### Courses Overview
- List of instructor's courses
- Enrollment count per course
- Course status (published, draft)
- Action buttons (edit, analytics, manage)

### Recent Submissions
- Recently submitted assignments
- Pending grading queue
- Submitted work awaiting feedback
- Quick grade/feedback buttons

### Student List
- List of enrolled students
- Student progress
- Performance statistics
- Filter by course
- Contact student button

### Course Analytics
- Total enrollments
- Student engagement metrics
- Quiz average scores
- Assignment completion rate
- Forum activity (if applicable)

### Quick Actions
- Create new course
- Edit course
- Grade submissions
- Send announcements
- View course performance

## Navigation

- Sidebar or tabs for different sections
- Course selector/switcher
- Date range filters
- Export data options

## Key Features

- Course creation wizard
- Bulk grading
- Student communication
- Performance analytics
- Student engagement tracking
