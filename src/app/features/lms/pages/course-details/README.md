# Course Details Page

## Overview
Detailed course information page with enrollment option.

## Functionality

- Display course information
- Show course syllabus/modules
- Display instructor profile
- Show student reviews and ratings
- Enrollment button
- Price information

## Sections

### Course Header
- Course image/banner
- Course title
- Instructor name and profile
- Star rating and reviews count
- Price

### Course Overview
- Course description
- Learning outcomes
- Course level
- Language
- Duration/weeks

### Syllabus
- List of modules
- List of lessons per module
- Estimated time per lesson
- Lesson types (video, reading, quiz)

### Instructor Info
- Instructor name
- Instructor bio
- Instructor rating
- Instructor courses

### Reviews
- Student reviews and ratings
- Filter/sort reviews
- Show total reviews

### Enrollment Section
- "Enroll" button
- Price display
- "Add to Cart" button
- Course access terms

## Interactions

- Click "Enroll" → Enroll in course
- Click "Add to Cart" → Add course to cart
- View instructor profile
- Read and filter reviews

## For Enrolled Users

- Show "Continue Learning" instead of "Enroll"
- Show progress bar
- Show current section
