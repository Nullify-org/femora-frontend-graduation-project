# Assignment Submission Page

## Overview
Interface for submitting coursework assignments.

## Functionality

- Display assignment description
- File upload capability
- Text submission area
- Previous submissions list
- Submission deadline
- Grading feedback

## Assignment Info

- Assignment title
- Description and requirements
- Deadline date/time
- Points/grading scale
- Rubric (if applicable)
- Additional resources

## Submission Form

- **File Upload**: Drag-and-drop or file picker
- **Text Editor**: Rich text editor for text submissions
- **Comments**: Optional submission comments
- **Submit Button**: Submit assignment

## Submission Validation

- Check file size
- Check file type
- Validate before deadline
- Show validation errors

## Previous Submissions

- List all previous submissions
- Submission date/time
- Submission status (pending, graded)
- Grades/feedback if available
- Option to resubmit

## Grading Display

- Grade/score
- Grader feedback
- Rubric breakdown (if applicable)
- Comments from instructor
- Re-submission option if available

## Mobile Support

- Touch-friendly file upload
- Mobile text editor
- Responsive layout
- Easy resubmission
