# Trainee Profile Page

## Overview
Student/trainee profile showing learning progress and achievements.

## Functionality

- Display student information
- Learning goals and interests
- Current courses and progress
- Completed courses and certificates
- Learning statistics
- Skill badges

## Sections

### Profile Info
- Name and profile picture
- Bio/about me
- Learning goals
- Interests (from onboarding)

### Learning Progress
- Current courses with progress bars
- Time spent learning
- Assignments submitted
- Quizzes taken

### Achievements
- Completed courses
- Certificates earned
- Skill badges
- Milestones reached

### Statistics
- Total hours learned
- Courses completed
- Average score
- Learning streak

## Edit Mode

- Edit profile information
- Update learning goals
- Update interests
- Change profile picture
