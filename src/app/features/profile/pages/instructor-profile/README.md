# Instructor Profile Page

## Overview
Instructor profile and teaching credentials management.

## Functionality

- Display instructor information
- Teaching expertise
- Qualifications and certifications
- Teaching experience
- Course statistics
- Student reviews
- Video introduction

## Sections

### Personal Info
- Name and bio
- Profile picture
- Contact information

### Expertise
- Subject areas (multi-select)
- Specializations
- Years of experience

### Qualifications
- Degree/Education
- Certifications
- Professional credentials

### Teaching Stats
- Total students
- Total courses
- Courses published
- Average rating
- Student reviews

### Video Introduction
- Video upload/preview
- Video description

## Edit Mode

- Edit personal information
- Add/edit qualifications
- Upload video introduction
- Form validation
- Save/Cancel buttons
