# Preferences Page

## Overview
User preferences and account settings page.

## Functionality

- Manage privacy settings
- Notification preferences
- Email subscriptions
- Language and timezone
- Account security
- Theme preferences

## Settings Sections

### Privacy
- Profile visibility (Public/Private/Friends only)
- Show email in public profile
- Allow direct messages
- Share activity status

### Notifications
- Email notifications toggle
- In-app notifications toggle
- Notification types (courses, messages, promotions)
- Notification frequency

### Email Preferences
- Course updates
- Marketplace notifications
- Promotions and offers
- Weekly digest
- Unsubscribe link

### System Preferences
- Language selection
- Timezone
- Date format
- Theme (Light/Dark/Auto)

### Account Security
- Password change
- Two-factor authentication
- Login history
- Active sessions

## Features

- Toggle switches for on/off settings
- Dropdown selections
- Save preferences button
- Confirmation messages
- Reset to defaults option
