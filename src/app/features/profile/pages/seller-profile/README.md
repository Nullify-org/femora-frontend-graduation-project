# Seller Profile Page

## Overview
Seller profile and store management page.

## Functionality

- Display seller information
- Store name and description
- Business categories
- Review/rating statistics
- Store policies
- Payment account info
- Edit capability

## Sections

### Store Info
- Store name
- Store description
- Store logo/banner
- Categories (multi-select)

### Performance
- Total sales
- Average rating
- Customer reviews
- Response rate

### Policies
- Shipping policy
- Return policy
- Payment methods

### Payment Settings
- Bank account details
- Payment method setup
- Withdrawal settings

## Edit Mode

- Toggle edit mode button
- Form validation
- Save/Cancel buttons
- Show success/error messages
