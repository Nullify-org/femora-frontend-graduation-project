# Product Catalog Page

## Overview
Browse and discover products in the marketplace.

## Functionality

- Display all products
- Search products
- Filter by category, price, rating
- Sort products
- Pagination or infinite scroll
- Product cards

## Product Card Information

- Product image
- Product name
- Price
- Star rating
- Number of reviews
- Seller name
- Stock status
- Quick add to cart

## Filters

- **Category**: Select from categories
- **Price Range**: Min and max price
- **Rating**: Minimum rating
- **Condition**: New, Used, Refurbished
- **Seller**: Filter by seller

## Search

- Real-time search
- Search in title and description
- Suggest popular searches

## Sorting

- Price: Low to High, High to Low
- Rating: Highest first
- Newest: Most recent
- Popular: Most purchased

## Interactions

- Click product → Navigate to product-details
- Quick add to cart → Add item
- Save to wishlist → Save item
- View seller → See seller profile

## Mobile Responsive

- Single column on mobile
- Filter drawer on mobile
- Touch-friendly interactions
- Fast loading
