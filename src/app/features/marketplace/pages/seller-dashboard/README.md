# Seller Dashboard Page

## Overview
Dashboard for sellers to manage products and orders.

## Functionality

- Manage products
- View orders
- Track sales
- Manage inventory
- Process payments
- Customer communication

## Dashboard Sections

### Products Management
- Product list
- Product status (active, draft, archived)
- Create new product
- Edit product
- Delete product
- Inventory management
- Product performance metrics

### Orders
- Recent orders
- Order status tracking
- Order details
- Print shipping label
- Mark as shipped
- Process returns

### Sales Analytics
- Total sales
- Revenue charts
- Top selling products
- Sales by category
- Customer acquisition
- Average order value

### Inventory
- Stock levels
- Low stock alerts
- Reorder management
- Inventory forecast

### Customer Communication
- Messaging interface
- Order inquiries
- Shipping questions
- Return requests

### Store Settings
- Store information
- Shipping policies
- Return policies
- Payment settings

## Navigation

- Sidebar with main sections
- Quick action buttons
- Date range filters
- Export data options

## Key Features

- Bulk product operations
- Sales forecasting
- Automated low stock alerts
- Customer feedback
- Performance analytics
